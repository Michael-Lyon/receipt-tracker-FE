# Receipt Tracker Frontend

Smart Receipt Expense Tracker with AI-powered OCR capabilities - Frontend application built with React and Vite.

## Features

- User authentication and registration
- Receipt image upload and processing
- AI-powered OCR for automatic data extraction
- Expense dashboard and management
- Receipt editing and categorization

## Technology Stack

- **React** - Frontend framework
- **Vite** - Build tool and development server
- **Chakra UI** - Component library
- **React Router** - Navigation
- **Axios** - HTTP client

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) to view the application

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
